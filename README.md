## 📖 О проекте

Приложение ищет жителей жилого дома по имени и фамилии и возвращает дату рождения и возраст. Данные хранятся в многоуровневой JS-структуре: подъезд → этаж → квартира.

---

## 📁 Структура проекта

```
birthday/
├── index.html      ← разметка страницы
├── script.js       ← данные и логика поиска
├── style.css       ← стили (тёмная тема, золото)

└── favicon.jpg
```

Никаких зависимостей. Просто открой `index.html` в браузере.

## 🏠 Структура данных

Объект `MyHome` описывает здание из **5 подъездов**, в каждом по **5 этажей** с одной квартирой. Итого **25 жильцов**.

```js
const MyHome = {
  entranceOne: {
    floorOne: {
      apartmentOne: {
        fullName: 'Александр Петров',
        birthDay: '12.03.1985 - 41'
      }
    }
    // ...
  }
  // entranceTwo ... entranceFive
}
```

| Уровень | Ключи |
|---------|-------|
| Подъезд | `entranceOne` — `entranceFive` |
| Этаж | `floorOne` — `floorFive` |
| Квартира | `apartmentOne` — `apartmentTwentyfive` |

## ⚙️ Как работает

### Логика — функция `search()`

```
1. Считать имя и фамилию из полей ввода
2. Собрать строку: firstName + ' ' + lastName
3. Валидация:
   — поля не пустые
   — имя не содержит цифр (/\d/.test())
4. Тройной цикл for...in по MyHome:
   подъезд → этаж → квартира
   если person.fullName === searchFullName (без учёта регистра)
   → foundPerson = person
5. Показать результат или сообщение об ошибке
6. Очистить поля ввода
```

---

## ✅ Валидация

| Ситуация | Результат |
|----------|-----------|
| Поля пустые | `Ошибка ввода` — заполните имя и фамилию |
| Есть цифры в имени | `Ошибка ввода` — имя не может содержать цифры |
| Человек найден | Имя, дата рождения и возраст |
| Человек не найден | `К сожалению ничего не найдено` |
